# Bookmarkd

A social reading platform where users can track books, write reviews, and interact with other readers.

---

## Overview

Bookmarkd is a full-stack web application built with React and GraphQL. It allows users to manage personal reading lists, publish reviews, and interact with a community of readers.

The project emphasizes authentication flows, relational-style data modeling with MongoDB, and structured GraphQL APIs.

---

## Core Features

- User authentication (JWT)  
- Book search via Google Books API  
- Reading list tracking (To-Read, Currently Reading, Finished)  
- Review creation and rating  
- User profiles  
- Book club functionality  

---

## Tech Stack

- React (Vite)  
- Apollo Client & Server (GraphQL)  
- Node.js & Express  
- MongoDB with Mongoose  
- Google Books API
